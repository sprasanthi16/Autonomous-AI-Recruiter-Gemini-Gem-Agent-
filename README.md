# Autonomous-AI-Recruiter-Gemini-Gem-Agent-
An automated AI Agent built on Google’s Gemini 1.5 Pro architecture designed to perform high-fidelity Gap Analysis for job seekers. The agent uses a "Gatekeeper" protocol to ensure data integrity before performing a multi-pillar vetting process against 2026 market standards.
Key Features:

Gatekeeper Protocol: A custom-built logical sequence that prevents analysis until three validated inputs (Resume, Role, Salary) are provided.  

Automated Validation: Checks for ATS-compliant file formats (.pdf, .docx) and identifies standard resume headers via Python-based parsing.  

Gap Analysis Engine: Cross-references candidate experience against target salary benchmarks to produce a "Path to Unrejectable" report.  

Visual Markdown Reporting: Renders a high-fidelity report in-chat featuring a compatibility score ring, skill assessment tables, and a 90-day roadmap.
